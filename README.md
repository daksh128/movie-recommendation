# movie-recommendation

🎬 Movie Recommendation System

📌 About

This project is a simple Movie Recommendation System built using Python.
It suggests similar movies based on your favorite movie.

🛠️ Tools Used
Python
Pandas
NumPy
Scikit-learn
▶️ How to Run
Install libraries:
pip install numpy pandas scikit-learn
Run the project:
python movie.py
Enter your favorite movie name and get recommendations.
📊 Features
Finds similar movies
Uses TF-IDF and cosine similarity
Suggests top 20 movies
📂 Files
movie.py → main code
movies.csv → dataset

Enter your favourite movie name: Avatar
